This is a quick tool to lookup words, names and places while reading a book. 

To use the tool, add the file to your bin and path. Make it executable.

You will need 

To use:
book LOOKUP
Where LOOKUP is the name, place or person you want to look up. 

The program will check your entry against a dictionary, and if that fails then against wikipedia. If the dictionary lookup is succesful, it returns the definition of the word you looked up. If the wikipedia lookup is succesful, it returns the summary.
